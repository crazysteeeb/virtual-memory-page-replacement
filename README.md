# virtual-memory-page-replacement
C++ simulation of virtual memory paging and page replacement concepts for operating systems coursework.

## Overview

This project simulates how an operating system manages virtual memory pages and handles page replacement operations.

The simulator demonstrates core memory management concepts such as:
- Virtual memory
- Paging
- Page replacement
- Memory allocation behavior
- Operating systems memory management

## Technologies Used

- C++
- Linux/Ubuntu
- Visual Studio Code
- Command-line development environment

## Project Structure

```text
main.cpp      -> Program entry point
pager.cpp     -> Paging and memory management logic
